Recam치n Sequence is a numeric one. This sequence is particularly beutiful and interesting as a kind of strange balance between order and chaos. There are two ways of discovernig the balance. The first one is to try to visualize it and the second one is to try to interpret it with music.

Visualization

Let's say we have a counter. It's going to start at 1 and then go up (1, 2, 3, ...). For every counter value we're going to try to go backwards by couter value steps. If the spot is taken or there are no numbers backwards, we go forward.

So, we start at 0 with counter value = 1. We're trying to go backwards, and.. Oh, no!!! There are no numbers in the sequence :( But we don't give up and go to 1.

Now we're trying to go backwards by 2. Unfortunately, we can't. So, we go forward by 2 and land on 3.

Now we want to go backwards by 3. But the spot is taken :^( So we have no alternative but to go to 6.

Far from losing hope, we should face the challenges and try to go backwards by 4. And yay! The spot is open, so we land on 2.

Now we try to go backwards by 5. Well, we can't, so we go to 7.


And so on...
